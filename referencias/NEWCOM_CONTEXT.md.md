Contexto del Proyecto: Rediseño Newcom Argentina (Next-Gen ITS)
1. Visión General
Cliente: Newcom Argentina. Objetivo: Transformar una web institucional tradicional en una plataforma de alto impacto visual y tecnológico. Estética: "Industrial Intelligence" (Estilo Palantir, Tesla). Dark Mode absoluto, minimalismo extremo, tipografía técnica y sensaciones de "Centro de Control" o "Soberanía Tecnológica".

2. Definición del Producto (ITS & Smart Cities)
El enfoque central es ITS (Intelligent Transport Systems): soluciones basadas en TIC para un transporte más seguro, eficiente y sostenible.

Misión: Gestionar el tráfico, mejorar la movilidad y facilitar la conducción automatizada.

Seguridad Pública: Integración de videovigilancia inteligente y analíticas para entornos urbanos.

3. Pilares de Contenido (Core Solutions)
El desarrollo debe centrarse en estas verticales estratégicas:

LPR & Anillo Digital:

Sistemas de lectura de patentes en tiempo real para autopistas (Caso de éxito: AUSA/AUBASA).

Creación de perímetros digitales para control de ingresos y egresos urbanos.

Pesaje Dinámico (WIM - Weigh In Motion):

Balanza inteligente que pesa camiones a velocidad de autopista sin detener el flujo.

Cobro dinámico y desvío automático de unidades con exceso de carga.

Cybersecurity & Infrastructure:

Protección de infraestructura crítica y datos de transporte.

Implementación de protocolos de seguridad para evitar vulnerabilidades en Smart Cities.

Software Foundry:

Desarrollo de capas de software a medida, dashboards de control y procesamiento de datos críticos en tiempo real.

4. Guía de Diseño (UI/UX)
Paleta de Colores: * Fondo: Negro Puro (#000000) o Carbón profundo (#050505).

Acentos: Azul Eléctrico (#0066FF) o Cyan Neón.

Bordes: white/10 o white/5 para una estética glassmorphism sutil.

Tipografía: * Inter o Geist para el cuerpo y títulos (limpieza absoluta).

JetBrains Mono para datos técnicos, labels y métricas (aire de ingeniería/código).

Layout: One-page con scroll suave. Uso de Bento Grids para las soluciones y secciones de Technical Performance con métricas animadas.

5. Referencias de Contenido y Estructura
Se deben utilizar conceptos de seguridad y monitoreo urbano de las siguientes fuentes líderes:

Sistemas ITS: Gestión de movilidad y seguridad vial.

Ciberseguridad: Protección de activos tecnológicos urbanos.

Anillo Digital: Control de patentes y seguridad perimetral.

Seguridad Pública: Despliegue de tecnología para prevención del delito.

6. Stack Tecnológico Sugerido
Framework: React / Next.js (o HTML estático según preferencia).

Styling: Tailwind CSS (utilizando el archivo de configuración provisto).

Icons: Lucide React o Material Icons (estilo outlined).

Animations: Framer Motion (para fade-ins y contadores de datos).
